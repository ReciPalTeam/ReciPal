## Feature: Add Quantity Selector to Manual Pantry Item Add

### What to build
When a user manually adds an item to their pantry, a quantity selector needs to appear in the item row — in the space between the item name/category label and the expiration date pill. The unit auto-selects based on the ingredient name using colloquial grocery units (e.g., "gallon" for milk, "lb" for chicken, "dozen" for eggs, "can" for canned beans).

### Scope
This is ONLY for the manual add flow. Do not modify receipt scan, purchase, or meal-cooked deduction logic.

### 1. Data model changes

**`client/src/lib/demo-store.ts`:**
- Add `quantity: number` and `unit: string` to the `PantryItem` interface (lines 157-167)
- Update `addToPantry` action (line 590) to accept and store `quantity` and `unit`
- Update `createPantryItem()` helper (line 479) to include quantity/unit
- Update `INITIAL_PANTRY` seed items (line 500) with realistic quantities and units

**`shared/schema.ts`:**
- Add `quantity` (doublePrecision, default 1) and `unit` (text, default 'each') to the `pantryItems` table (line 193)

### 3. UI — Quantity Selector

**Position:** Between the item name/category text and the expiration date pill in the pantry item row.

**Layout:** Horizontal inline: `[-] [quantity] [+] [unit label]`

**Behavior:**
- `[-]` and `[+]` adjust by the `step` value for that unit
- Cannot go below `min` for that unit
- No maximum
- Tapping the unit label opens a small dropdown/popover to switch to an alternate unit (from the food group's alternate units list). When switching units, reset quantity to 1.
- Compact styling — `text-xs`/`text-sm`, tight padding, fits within the existing card layout

**Pattern to follow:** Reuse the +/- stepper pattern from `client/src/components/manual-entry-sheet.tsx` lines 469-489.

### 4. Integration in Add Pantry Item dialog

In `client/src/pages/pantry/index.tsx` (the `<Dialog>` at line 417):
- When user types an item name, auto-detect unit via `getDefaultPantryUnit(name, foodGroup)` (same way food group is already auto-detected via `getIngredientFoodGroup()`)
- Add quantity state (`newItemQuantity`) and unit state (`newItemUnit`)
- Show the quantity selector below the Food Group select, before the Add button
- Pass quantity and unit into `addToPantry()` on confirm

### 5. Display in pantry item cards

In the pantry list cards (line 335+), show quantity + unit as a subtle label near the item name (e.g., "2 lb" or "1 dozen" below or beside the food group label).
