Goal:
 Create a segmented control with a flush, glass-style green selection indicator. The green selection must sit perfectly flush inside the pill container (no floating, no inset gaps), and it should move smoothly between tabs.
Structure requirements
Use one outer container (the track)
Use one absolute-positioned selection indicator inside the container
Use buttons for each segmentThe indicator must move by changing a CSS variable (not recreating elements)

Visual requirements
• The outer container:
Rounded pill shape
Light, frosted background
Subtle inner border
• The green active indicator:
Flush with the inner edge of the container
Translucent green “glass” look
Soft glow and subtle highlight
Uses backdrop-filter for glass effect
• Smooth animated transition when switching tabs

CSS rules you MUST follow
• Use position: relative on the container
• Use position: absolute on the active indicator
• Use CSS variables for:
active index
inset padding
• Use border-radius: inherit or a calculated radius so edges align perfectly
• Use rgba() colors (no solid hex greens)
• Use backdrop-filter: blur() and saturate()

Implement exactly this pattern

<div class="segmented" style="--active-index: 0;">
  <div class="segmented__thumb"></div>

  <button class="segmented__btn active">For You</button>
  <button class="segmented__btn">Something New</button>
  <button class="segmented__btn">Favorites (1)</button>
</div>

.segmented {
  --radius: 999px;
  --inset: 3px;
  --count: 3;
  --active-index: 0;

  position: relative;
  display: grid;
  grid-template-columns: repeat(var(--count), 1fr);
  padding: var(--inset);
  border-radius: var(--radius);

  background: rgba(255,255,255,0.55);
  border: 1px solid rgba(255,255,255,0.6);
  backdrop-filter: blur(10px);
}

.segmented__thumb {
  position: absolute;
  top: var(--inset);
  left: var(--inset);
  width: calc((100% - (2 * var(--inset))) / var(--count));
  height: calc(100% - (2 * var(--inset)));
  border-radius: calc(var(--radius) - var(--inset));
  transform: translateX(calc(var(--active-index) * 100%));
  transition: transform 200ms ease;

  background:
    linear-gradient(180deg, rgba(255,255,255,0.35), rgba(255,255,255,0.1)),
    rgba(34,197,94,0.55);

  border: 1px solid rgba(255,255,255,0.45);
  box-shadow:
    0 8px 18px rgba(34,197,94,0.25),
    inset 0 1px 0 rgba(255,255,255,0.6);

  backdrop-filter: blur(14px) saturate(160%);
}

Interaction behavior
• On button click:
Update --active-index
Toggle .active class on buttons
• Do not resize or recreate the thumb
• Do not use box-shadow hacks to fake positioning

Final acceptance criteria
The green indicator must look embedded, not floatingNo visible gap between indicator and container edgesSmooth horizontal motionGlassy, iOS-style feel
