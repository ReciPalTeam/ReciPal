D) GUIDE ME PATH (Mifflin–St Jeor)
────────────────────────────────────────

D1) Step 1: Goal
Question: “What’s your primary goal?”
Options (radio):
- Lose fat
- Maintain weight
- Build muscle
- Improve performance / energy

D2) Step 2: Sex
Radio: Male / Female

D3) Step 3: Height (US Imperial default)
Inputs: feet + inches

D4) Step 4: Weight (US Imperial default)
Input: pounds

D5) Step 5: Activity level (with definitions inline)
Radio options:
- Light — Works out or participates in physical activities casually 1–2 times per week
- Moderate — Works out or participates in physical activities 3–4 times per week
- Very Active — Works out or participates in physical activities 5+ times per week

D6) Calculation rules (deterministic)
- Convert lbs→kg, ft/in→cm
- BMR via Mifflin–St Jeor:
  Male: BMR = 10*w + 6.25*h − 5*a + 5
  Female: BMR = 10*w + 6.25*h − 5*a − 161
NOTE: Age is not collected in your current required list. Since your spec did NOT include age, do NOT add it here.
Instead:
- Use a reasonable default age constant (e.g., 30) ONLY if you must compute BMR,
- OR implement a simplified calorie estimate not requiring age.
Preferred approach:
- Add a single required input “Age” ONLY if already present in the app.
If Age is NOT present anywhere, do NOT introduce it in this prompt. Keep it consistent with your defined 5-core questions.

Activity multipliers:
- Light: 1.375
- Moderate: 1.55
- Very Active: 1.725

TDEE = BMR * multiplier

Goal adjustments:
- Lose fat: 15% deficit
- Maintain: 0%
- Build muscle: 10% surplus
- Improve performance/energy: 5% surplus (or maintain + small surplus)

Protein prioritization:
- Lose fat: 2.0 g/kg
- Maintain: 1.6 g/kg
- Build muscle: 1.8 g/kg
- Improve performance: 1.6 g/kg

Fat:
- 0.8 g/kg (ensure not extremely low)

Carbs:
- remainder calories after protein + fat

Convert calories ↔ grams using 4/4/9.
Round grams to whole numbers.

D7) Output targets:
- caloriesTarget
- proteinTarget_g
- carbsTarget_g
- fatTarget_g
