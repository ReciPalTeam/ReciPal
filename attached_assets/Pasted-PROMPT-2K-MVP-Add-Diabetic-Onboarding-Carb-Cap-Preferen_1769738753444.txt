PRIMARY GOAL
Add a safe “Diabetic?” question to onboarding and, if Yes, collect a user-controlled daily or per-meal carb cap (user sets it themselves). This is a preference setting, not a medical classification. We must avoid implying recipes are medically appropriate.

4) Wording must be neutral and user-controlled:
   - “Do you want to set a carb limit?” / “Carb limit (grams)” / “You control this setting.”
