5) Measurements policy (IMPORTANT):

   - If pantry category is “Produce” and unit is cup (ex: “cup, chopped”), omit measurement and send name only (Instacart will pick head/bag/bunch/etc).
   - Rounding:
     - If you DO send a countable “each” quantity and it is fractional (ex: 1.5 each), round UP to nearest whole number and log roundingApplied=true.
     - Do NOT round weight units (oz/lb/g/kg) unless absolutely required by Instacart (prefer leaving as-is).

F) Measurement policy implementation details (SO REPLIT DOESN’T GUESS):
Create a pure function in the Instacart integration layer:
- decideInstacartMeasurement(lineItem) -> { includeMeasurement:boolean, qty?:number, unit?:string, roundingApplied?:boolean, omittedReason?:string }
Rules:
1) If pantryCategory == "Spices & Seasonings":
   - includeMeasurement = false
   - omittedReason = "spice_container"
2) If pantryCategory == "Produce" AND (recipeUnit == "cup" OR originalUnitDisplay contains "cup"):
   - includeMeasurement = false
   - omittedReason = "produce_recipe_volume"
3) If recipeUnit in ["serving"] OR originalUnitDisplay contains "serving":
   - includeMeasurement = false
   - omittedReason = "serving_unit"
4) If originalUnitDisplay contains "(" or ")" OR contains "juice of" (case-insensitive):
   - includeMeasurement = false
   - omittedReason = "unsupported_unit_display"
5) If confidence == "low" AND recipeUnit not in ["each","ounce","pound","gram","kilogram","ml","liter","fl_oz"]:
   - includeMeasurement = false
   - omittedReason = "low_confidence_unit"
6) Otherwise:
   - If recipeUnit == "each":
       qty = recipeQty
       if qty is fractional => round up to integer, roundingApplied=true, omittedReason not set
       includeMeasurement=true, unit="each"
   - If recipeUnit in ["ounce","pound","gram","kilogram","ml","liter","fl_oz"]:
       includeMeasurement=true, qty=recipeQty, unit=recipeUnit
   - Else:
       includeMeasurement=false, omittedReason="other_recipe_unit"

NOTE: This means mayo “58 gram” will include measurement (fine), shrimp “4 ounce” includes measurement (fine), tortillas “each” includes, but cumin/oregano/dill (spices) omit, cabbage/cilantro measured in cups omit, lime “juice of …” omit.
